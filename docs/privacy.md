# 隐私政策与用户协议

> **生效日期：2026 年 5 月 1 日**  
> 本声明依据《中华人民共和国个人信息保护法》《网络安全法》《数据安全法》及推荐性国家标准 GB/T 35273-2020《信息安全技术 个人信息安全规范》制定。  
> [English Version](../privacy-en/) · [回到首页](../)

## 一、概述

海阅 (HLib) 是一款基于公开 Z-Library 镜像 API 的第三方阅读客户端，由独立开发者发布，与 Z-Library 官方无关联。本应用本身不存储、不分发任何书籍内容，所有内容来源于您选择的镜像站点。

本声明说明了我们如何收集、使用、存储和保护您的个人信息，以及您对个人信息享有的权利。请您仔细阅读并充分理解本声明各条款，特别是以**加粗**形式提示您注意的条款。

## 三、个人信息收集与使用

本应用仅在本地设备保存为提供核心功能所必需的数据，**不会上传至开发者服务器**：

### （一）账号信息（敏感，仅本地）

- Z-Library 账号 cookie（`remix_userid` / `remix_userkey`）——使用华为 HUKS 硬件级密钥加密存储，仅用于以登录身份请求镜像 API；
- 仅在您主动登录或扫码导入时收集，您可随时通过「退出登录」清除。

### （二）业务数据（仅本地）

- 收藏书目 ID 列表、最近搜索关键词；
- 下载任务、下载历史、阅读进度（仅文件名、URL、时间戳，**不含书籍内容**）。

### （三）应用偏好（仅本地）

- 主题模式、语言、镜像选择、代理与 DoH 设置、导航样式偏好。

> **所有网络请求直接发往您选择的 Z-Library 镜像域名，不经任何中间服务器。本应用不接入任何第三方统计、广告、推送、崩溃上报或埋点 SDK。**

## 四、权限说明

本应用按「最小必要」原则申请权限。每项权限的用途和触发时机：

| 权限 | 类型 | 用途 | 触发时机 |
|---|---|---|---|
| `ohos.permission.INTERNET` | 必需 | 访问 Z-Library 镜像、下载书籍 | 安装时授予 |
| `ohos.permission.GET_NETWORK_INFO` | 必需 | 检测连接可用性、提示离线状态 | 安装时授予 |
| 相机 | 可选 | 识别二维码以快捷导入凭证；图像仅用于本次解析，不保存、不上传 | 仅在您点击「扫码登录」时弹出 |
| 媒体读写 | 可选 | 将书籍保存到您指定的公共目录；本应用不会读取目录中其他文件 | 仅在您选择「公共下载目录」时弹出 |

您可随时在系统「设置 → 应用 → 海阅 → 权限」中关闭可选权限。

## 五、对外提供

本应用承诺：

- **不向任何第三方共享**您的个人信息；
- **不向任何第三方转让**您的个人信息；
- **不公开披露**您的个人信息；
- **不存在委托第三方处理**您的个人信息的情形。

您的所有数据仅在本设备本地存储与处理。本应用与镜像站点的交互仅限于您主动发起的查询/下载请求，**不会主动上报任何使用数据**。

---

## 六、信息保护措施

我们采取以下措施保护您的个人信息：

- 敏感凭证（账号 cookie）使用 HarmonyOS HUKS（硬件密钥库）加密后再写入 Preferences；
- 所有网络请求强制使用 HTTPS（除您手动配置的 HTTP 镜像或代理外）；
- 不收集任何超出业务功能必需的信息；
- 卸载应用时，操作系统会自动清除全部本地数据，**不存在残留**。

尽管我们采取了合理的安全措施，但请理解互联网环境并非绝对安全，您应妥善保管设备和账号。

## 七、您的权利

依据《个人信息保护法》和 GB/T 35273 国家标准，您对个人信息享有以下权利：

- **查阅与更正**：所有数据均在「设置 / 收藏 / 下载历史」等界面可视，可直接查看与修改；
- **删除**：在「设置 → 清除缓存」中可清除全部本地数据；卸载应用可彻底移除；
- **撤回同意**：您可在「设置 → 隐私政策与用户协议」页面再次查阅本声明，或卸载应用以撤回您的同意；
- **注销账号**：本应用不在开发者侧创建账号，您的 Z-Library 账号注销请通过 Z-Library 官网完成；
- **拒绝个性化推送**：本应用不进行任何个性化推送或自动化决策。
