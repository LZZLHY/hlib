# 海阅 (HLib)

> **HarmonyOS 第三方电子图书阅读客户端**  
> 基于 Z-Library 公开镜像 API · 完全本地存储 · 零三方 SDK

## 简介

海阅是一款基于公开 Z-Library 镜像 API 的第三方阅读客户端，由独立开发者发布，**与 Z-Library 官方无关联**。本应用本身不存储、不分发任何书籍内容，所有内容来源于用户选择的镜像站点。

## 核心特性

- **沉浸式阅读体验**：浮动胶囊导航 + 模糊顶栏，专为 HarmonyOS 6.x 沉浸光感设计
- **平板/折叠屏适配**：响应式布局，自动切换底栏 ↔ 侧栏导航
- **多镜像智能切换**：内置 50+ Z-Library 镜像测速 + 自动选优
- **隐私优先**：账号凭证 HUKS 硬件级加密，所有数据仅本地存储
- **零追踪**：不接入任何第三方统计、广告、推送、崩溃上报或埋点 SDK
- **多语言**：简体中文、English
